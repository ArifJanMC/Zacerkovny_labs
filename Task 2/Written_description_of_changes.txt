Между after_sNAT_get.png и before_sNAT_get.png видим следующую разницу:
До NAT'а TTL 64, после прохождения этого маршрутизатора - 63 (64 - 1).
src.ip до - 10.10.0.12, после - 94.198.16.3 (благодаря iptables -t nat -A POSTROUTING -o eth1 -j SNAT --to 94.198.16.3)
Checksum - логично, что она пересчитывается после подмены значений.
Source port - Изменился, т.к. хост, выполнивший NAT, открыл другой сокет для отправки запроса серверу и отправки ответу изначальному source по его source port.
